🌿 초록색 테마 포트폴리오 웹사이트 사용법 🌿

📋 프로젝트 개요
- React 기반의 반응형 포트폴리오 웹사이트
- 초록색 테마로 디자인된 모던한 UI
- 5개 섹션으로 구성: Home, About, Skills, Projects, Contact

🚀 실행 방법

1. 개발 서버 시작 (프론트엔드 + 백엔드 동시 실행)
   npm run dev

2. 개별 실행
   - 프론트엔드만: npm start
   - 백엔드만: npm run server

🌐 접속 주소
- 프론트엔드: http://localhost:3002
- 백엔드: http://localhost:3001

📱 주요 기능

🏠 Home 섹션
- 자기소개 및 프로필 카드
- 프로젝트 보기 및 연락하기 버튼
- 애니메이션 효과

👤 About 섹션
- 개인 소개 및 경력 정보
- 통계 정보 (경력, 프로젝트, 만족도)
- 교육 및 관심 분야

💻 Skills 섹션
- 프론트엔드, 백엔드, 도구별 스킬 분류
- 진행률 바로 시각적 표현
- 호버 효과

📂 Projects 섹션
- 프로젝트 카드 형태로 표시
- 기술 스택 태그
- Live Demo 및 GitHub 링크

📞 Contact 섹션
- 연락처 정보 (이메일, 전화, 위치)
- 소셜 미디어 링크
- 연락 폼

🎨 디자인 특징
- 초록색 테마 (#2e7d32, #4caf50)
- 그라데이션 배경
- 반응형 디자인
- 부드러운 애니메이션
- 모던한 카드 디자인

📱 반응형 지원
- 데스크톱, 태블릿, 모바일 최적화
- 햄버거 메뉴 (모바일)
- 터치 친화적 인터페이스

⌨️ 네비게이션
- 상단 고정 네비게이션 바
- 스크롤 시 활성 섹션 하이라이트
- 부드러운 스크롤 이동

🔧 커스터마이징
- src/App.js: 콘텐츠 수정
- src/App.css: 스타일 수정
- 개인 정보, 프로젝트, 스킬 등 업데이트

📁 파일 구조
├── src/
│   ├── App.js          # 메인 컴포넌트
│   ├── App.css         # 스타일시트
│   └── index.js        # 앱 진입점
├── server.js           # 백엔드 서버
├── package.json        # 의존성 관리
└── 사용법.txt          # 이 파일

🎯 사용 팁
1. 개인 정보를 실제 정보로 교체하세요
2. 프로젝트 섹션에 실제 프로젝트를 추가하세요
3. 스킬 진행률을 실제 수준에 맞게 조정하세요
4. 연락처 정보를 실제 정보로 업데이트하세요
5. 소셜 미디어 링크를 실제 링크로 변경하세요

✨ 추가 기능 제안
- 다크 모드 토글
- 다국어 지원
- 블로그 섹션 추가
- 포트폴리오 갤러리
- 다운로드 가능한 이력서

🔍 문제 해결
- 포트 충돌 시: 다른 포트 번호 사용
- 스타일 문제: 브라우저 캐시 삭제
- 서버 오류: node_modules 재설치
